# Getting Started with Radxa Zero 3W

## Flash OS
Use BalenaEtcher to flash official or Yocto-based images.

## UART Serial Console
Use FTDI 3.3V adapter. Connect:
- GND → GND
- TX → RX
- RX → TX

Example:
```bash
screen /dev/ttyUSB0 115200
```

## First Boot
Login with default user, update packages manually.
