# 🏷️ Hardware Revisions

As the Radxa Zero 3W matures, Radxa releases minor hardware revisions to improve stability or change components.

---

## Current Revisions

### **v1.1 / v1.12** (Latest)
* **SoC:** Rockchip RK3566
* **Wireless:** AIC8800 (Wi-Fi 6 / Bluetooth 5.4)
* **Notes:** This is the standard production model currently available. Most schematics and datasheets refer to v1.12.

## Model Variants
The Radxa Zero 3W is sold in several memory and storage configurations. All variants share the exact same PCB layout and component ecosystem:

* **RAM:** 1GB, 2GB, 4GB, or 8GB LPDDR4
* **Storage:** MicroSD-only (No eMMC) **OR** 8GB / 16GB / 32GB / 64GB eMMC onboard

> **Tip:** You can check your total RAM via the command line using `free -h` or `cat /proc/meminfo`.
