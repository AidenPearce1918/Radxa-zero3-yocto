# 🖥️ Hardware Overview

The Radxa Zero 3W is a compact, high-performance single-board computer (SBC) designed for modern embedded systems and maker projects.

---

## 🖌️ Board Layout

![Radxa Zero 3W Hardware Overview](img/radxa_zero_3w_hardware.png)

The board features a quad-core ARM Cortex-A55 processor, LPDDR4 RAM, WiFi 5, Bluetooth 5.0, USB ports, HDMI 2.0, and a 40-pin GPIO header compatible with Raspberry Pi accessories.
